---
title: Shortcuts
description: Keyboard shortcuts for using Everywhere
---

| Function | Windows / Linux | macOS | Note |
| :--- | :--- | :--- | :--- |
| **Switch Assistant** | `Ctrl` + `1`-`9`, `0` | `Cmd` + `1`-`9`, `0` | Or scroll on the icon in the bottom left |
| **Hide Window** | `Esc` | `Esc` | |
| **Pin Window** | `Ctrl` + `D` | `Cmd` + `D` | |
| **Send Message** | `Enter` | `Enter` | |
| **New Chat** | `Ctrl` + `N` | `Cmd` + `N` | Must be in an existing chat |
| **Toggle History** | `Ctrl` + `H` | `Cmd` + `H` | |
| **Toggle Tools** | `Ctrl` + `T` | `Cmd` + `T` | Requires assistant support |
