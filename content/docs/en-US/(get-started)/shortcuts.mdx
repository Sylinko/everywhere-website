| Function             | Windows / Linux       | macOS                | Note                                     |
| :------------------- | :-------------------- | :------------------- | :--------------------------------------- |
| **Switch Assistant** | `Ctrl` + `1`-`9`, `0` | `⌘` + `1`-`9`, `0` | Or scroll on the icon in the bottom left |
| **Hide Window**      | `Esc`                 | `Esc`                |                                          |
| **Pin Window**       | `Ctrl` + `D`          | `⌘` + `D`          |                                          |
| **Send Message**     | `Enter`               | `Enter`              |                                          |
| **New Chat**         | `Ctrl` + `N`          | `⌘` + `N`          | Must be in an existing chat              |
| **Toggle History**   | `Ctrl` + `H`          | `⌘` + `H`          |                                          |
| **Toggle Tools**     | `Ctrl` + `T`          | `⌘` + `T`          | Requires assistant support               |

<Callout type="info" title="Copilot Shortcut">
For Windows users, if your keyboard has a `Copilot` key, you can use it as a shortcut to bring up the chat window.
</Callout>
