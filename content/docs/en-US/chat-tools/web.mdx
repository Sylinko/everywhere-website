## Settings

### Show Browser in Foreground

When enabled, the browser will be displayed while extracting webpage content, which is generally not recommended. A restart is required for changes to take effect.

**Default**: Disabled

### Open Browser

Opens an instance of the web browser used for extracting webpages. You can log in to frequently used sites in advance to reduce the chances of being flagged as a bot during extraction. For compatibility, the first launch may automatically download the web browser, which may take some time depending on network conditions.
