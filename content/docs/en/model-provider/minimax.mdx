This tutorial will guide you step-by-step on how to get an API key for [MiniMax](https://www.minimaxi.com/).

<Callout type="warning" title="Site">
In the preset mode, Everywhere will connect to the China mainland site of MiniMax. You need to visit [platform.minimaxi.com](https://platform.minimaxi.com) to get an API key.

If you need to access the global site, please switch to advanced mode and fill in:
- URL: `https://api.minimax.io/anthropic`
- API Schema: Anthropic
</Callout>

## Steps

- Go to the [MiniMax Open Platform](https://platform.minimaxi.com/login) and register & log in to your account.
- After logging in, click on [接口密钥](https://platform.minimaxi.com/user-center/basic-information/interface-key) on the left side of the page.

<HorizontalImg
    src="/model-provider/minimax/platform-api-keys.webp"
    alt="API keys page"
  />

- On the right side of the page, click `创建新的 API Key`, and then fill in the name of the key in the input box below to help you remember its purpose.

<HorizontalImg
    src="/model-provider/minimax/platform-create-api-key.webp"
    alt="Create API key"
    width="400px"
  />

- Click the **Create API Key** button, and your API key will be displayed. Copy this key to continue in Everywhere.

<HorizontalImg
    src="/model-provider/minimax/platform-generate-api-key.webp"
    alt="Generate API key"
    width="400px"
  />

<Callout type="info">
If you accidentally close the dialog box, you can click the copy button on this key in the [接口密钥页面](https://platform.minimaxi.com/user-center/basic-information/interface-key) to copy it.
</Callout>

<Callout type="warning" title="Privacy">
Please note that the API key is sensitive information. Do not disclose it to anyone or share it in public.
</Callout>
