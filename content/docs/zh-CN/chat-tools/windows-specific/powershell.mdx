## 注意

处于安全考虑，Everywhere 不允许 PowerShell 脚本运行的自动批准。每当助手尝试执行脚本时，您都需要手动批准该操作。请务必仔细审查每个请求，以确保它们来自可信的来源并且不会对您的系统造成损害。
