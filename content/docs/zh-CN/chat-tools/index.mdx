Everywhere 支持 **内置** 和基于 **MCP (Model Context Protocol)** 协议的工具。

## 如何使用内置工具

内置工具随 Everywhere 本体一同分发。

1.  在 **聊天工具** 页面，勾选你想要启用的工具。

<HorizontalImg
    src="/chat-tools/index/builtin-zh-CN.webp"
    alt="内置工具列表"
    width="450px"
  />

2.  在与模型聊天时，**请确保**：
    *   你使用的模型支持工具调用（Tool Calling / Function Calling）。
    *   在聊天窗口中启用了 **工具调用** 选项。

<HorizontalImg
    src="/chat-tools/index/chat-window-zh-CN.webp"
    alt="聊天窗口启用工具调用"
    width="300px"
  />

3.  当你发送的消息触发了工具的功能时，模型 **可能** 会尝试调用你启用的工具来回答你的问题。

<Callout type="warning" title="限制">
受限于模型能力，工具调用可能不会总按预期工作。例如，模型可能会选择不调用工具，或者在调用时提供了错误的参数。
</Callout>

## 贡献内置工具

我们欢迎熟悉 C# 开发的开发者为 Everywhere 贡献更多的内置工具。如果你有好的想法或已经实现了一个工具，请通过[提交 Pull Request](https://github.com/DearVa/Everywhere/pulls)的方式与我们分享。

## 未来的工具系统

我们理解不是所有人都熟悉 C# 开发。因此，我们计划在未来推出一个更简易的工具系统，让更多的开发者甚至是没有编程基础的用户，都能够方便快捷地编写和使用自己的工具。
