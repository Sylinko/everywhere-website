## 官方支持

所有在本章节侧边栏中列出的模型提供商均由 Everywhere 团队进行维护和支持并出现在 **预设模式** 的列表中。

我们致力于确保这些提供商与 Everywhere 的集成能够稳定、高效地运行。

## 云服务

我们推荐您通过云服务以使用 **原生模式** 的方式使用我们提供的模型，无需任何本地部署或配置。

前往 <DynamicLink href="/[lang]/docs/quick-start">快速开始</DynamicLink> 页面了解更多详情。

## 提供商的收录

为了优化开发资源并确保最佳的用户体验，我们的开发团队在选择支持哪些模型提供商时会进行主观评估。评估标准包括但不限于提供商的知名度、市场流行程度以及社区评价。

请放心，官方收录的模型提供商列表不会频繁发生变动，以保证服务的稳定性。

## 关于第三方服务

请注意，Everywhere 仅对官方直接支持的模型提供商提供技术支持。由于兼容性适配需要投入大量的开发时间，我们无法受理因使用第三方中转或代理 API 而产生的任何问题。

如果您在使用受支持的模型提供商时遇到问题，请在[GitHub 提出问题](https://github.com/Sylinko/Everywhere/issues)以寻求帮助。
