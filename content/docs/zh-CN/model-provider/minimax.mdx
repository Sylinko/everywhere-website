本教程将一步步指导您如何获取 [MiniMax](https://www.minimaxi.com/) 的 API 密钥。

<Callout type="warning" title="站点">
在预设模式下，Everywhere 会接入 MiniMax 的中国大陆站点，您需要访问 [platform.minimaxi.com](https://platform.minimaxi.com) 来获取 API 密钥。

如果需要接入全球站点，请切换至进阶模式并填入：
- URL: `https://api.minimax.io/anthropic`
- API 协议: Anthropic
</Callout>

## 步骤

- 访问[MiniMax 开放平台](https://platform.minimaxi.com/login)，并注册&登录账户。
- 登录后，在页面左方点击[接口密钥](https://platform.minimaxi.com/user-center/basic-information/interface-key)。

<HorizontalImg
    src="/model-provider/minimax/platform-api-keys.webp"
    alt="API keys 页面"
  />

- 在页面右侧点击`创建新的 API Key`，随后于输入框中填写密钥的名称以帮您记住它的用途。

<HorizontalImg
    src="/model-provider/minimax/platform-create-api-key.webp"
    alt="创建 API key"
    width="400px"
  />

- 点击**创建密钥**按钮，成功后会显示您的 API 密钥。将此密钥复制到 Everywhere 继续。

<HorizontalImg
    src="/model-provider/minimax/platform-generate-api-key.webp"
    alt="生成 API key"
    width="400px"
  />

<Callout type="info">
如果您不小心关闭了对话框，可以在[接口密钥界面](https://platform.minimaxi.com/user-center/basic-information/interface-key)点击此密钥的复制按钮来复制它。
</Callout>

<Callout type="warning" title="隐私">
请注意，API 密钥是敏感信息，请不要将其泄露给任何人或在公共场合分享。
</Callout>
