---
title: 快捷键
description: Everywhere 常用快捷键列表
---

| 功能 | Windows / Linux | macOS | 备注 |
| :--- | :--- | :--- | :--- |
| **切换助手** | `Ctrl` + `1`-`9`, `0` | `Cmd` + `1`-`9`, `0` | 也可鼠标滚轮滚动左下角图标 |
| **隐藏窗口** | `Esc` | `Esc` | |
| **置顶固定** | `Ctrl` + `D` | `Cmd` + `D` | |
| **发送消息** | `Enter` | `Enter` | |
| **新建对话** | `Ctrl` + `N` | `Cmd` + `N` | 需在已有对话中 |
| **切换历史页** | `Ctrl` + `H` | `Cmd` + `H` | |
| **工具调用开关** | `Ctrl` + `T` | `Cmd` + `T` | 需助手支持 |
