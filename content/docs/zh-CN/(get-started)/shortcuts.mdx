title: 快捷键
icon: SquareSlash
---

| 功能             | Windows / Linux      | macOS                | 备注                      |
| :-------------- | :-------------------- | :------------------- | :----------------------- |
| **切换助手**     | `Ctrl` + `1`-`9`, `0` | `⌘` + `1`-`9`, `0` | 也可鼠标滚轮滚动左下角图标 |
| **隐藏窗口**     | `Esc`                 | `Esc`                |                          |
| **置顶固定**     | `Ctrl` + `D`          | `⌘` + `D`          |                          |
| **发送消息**     | `Enter`               | `Enter`              |                          |
| **新建对话**     | `Ctrl` + `N`          | `⌘` + `N`          | 需在已有对话中            |
| **切换历史页**   | `Ctrl` + `H`          | `⌘` + `H`          |                          |
| **工具调用开关** | `Ctrl` + `T`          | `⌘` + `T`          | 需助手支持                |

<Callout type="info" title="Copilot 快捷键呼出">
对于 Windows 用户，如果您的键盘上有`Copilot`键，可以将其作为呼出聊天窗口的快捷键。
</Callout>
