## 系统环境

目前，Everywhere 支持 Windows 和 macOS 平台。

<Callout type="info" title="系统要求">
- Windows 版本支持 Windows 10 (19041) 及更高版本。
- macOS 版本支持 macOS Monterey 12.0 及更高版本。
</Callout>

<Callout type="warning" title="Linux 支持">
Linux 版本由社区维护并开发中。
</Callout>

## Windows 安装说明

您可以下载到安装版(`.exe`)或便携版(`.zip`)，两个版本除了打包方式不同外无其他区别。

- 安装版的文件名是`Everywhere-Windows-x64-Setup-v<版本号>.exe`，在下载后需要双击运行安装向导，如果遇到 UAC (用户帐户控制) 提示，请选择“是”以允许安装程序继续运行。
- 便携版的文件名是`Everywhere-Windows-x64-v<版本号>.zip`，文件下载后需要解压缩到一个文件夹中，然后直接运行其中的`Everywhere.Windows.exe`即可。

<Callout type="info" title="数据存储位置">
无论是安装版还是便携版，Everywhere 的用户数据（如设置、聊天历史等）都会存储在用户的应用数据目录中，通常位于`C:\Users\<用户名>\AppData\Roaming\Everywhere`。
</Callout>
