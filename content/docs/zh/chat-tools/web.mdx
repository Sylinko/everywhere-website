## 设置

### 在前台显示浏览器

开启后，提取网页内容时将会显示浏览器，通常不建议开启。需重启软件生效。

**默认**：关闭

### 打开浏览器

打开一个提取网页所用的网络浏览器实例，可以提前登录常用网站以减少提取网页时被当做机器人而拦截的机率。出于兼容性考虑，首次打开可能会自动下载网络浏览器，视网络情况可能需要较长的时间。
