## 设置

### 自动批准 <PermissionBadge risk="high">高风险</PermissionBadge>

开启后，所有终端操作将自动放行，无需手动批准。助手的不当调用可能会造成严重的系统崩溃或者不可恢复的文件丢失。**开启此选项意味着你已评估并愿意自行承担风险。**

**默认**：关闭
