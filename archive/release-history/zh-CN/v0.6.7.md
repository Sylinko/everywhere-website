## ✨ 新特性
- 核心能力突破：加入了 `get_visual_tree` 工具。助手现在可以主动调用此工具自由获取屏幕上的元素节点，这为其赋予了远超以往的上下文理解能力与深度的自我感知能力。提示：此为实验性功能，可能无法在所有场景正常工作，欢迎提出改进意见！
- 模型更新：下架了即将过期的 `Gemini 3 Pro Preview` 模型，并新增了最新的 `Gemini 3.1 Pro Preview` 模型。

## 🚀 改进
- 基础组件升级：更新了底层 MCP (Model Context Protocol) 框架的版本，这有望从底层解决相关的功能异常反馈 (#232)。

## 🐛 修复
- **(重大修复)** 修复了 Windows 平台提取视觉上下文时的算法致命错漏。该缺陷曾导致在解析复杂界面时内容被过早截断，使助手实际上“看不见”所选元素下方的重要有效信息。
- 修复了开启屏幕划词功能后，在特定场景下可能触发应用渲染崩溃的问题 (#313)。
- 修复了由底层 Grid 布局计算偶然引发的随机崩溃 Bug，全方位提升了软件的稳定性。
